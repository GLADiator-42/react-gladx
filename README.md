```bash
npm install --save react-gladx
```

## Usage

Wrap your app in the `GladX` component.
```jsx
import React, { Component } from 'react'

import MyComponent from 'react-gladx'

class Example extends Component {
  render () {
    return (
      <MyComponent />
    )
  }
}
```
